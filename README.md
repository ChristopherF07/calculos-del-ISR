# Calculadora de ISR en C++

## Resumen

Este programa fue desarrollado en C++ con el propósito de calcular el Impuesto Sobre la Renta (ISR) de un empleado en la República Dominicana utilizando la escala salarial vigente para el año 2026 publicada por la DGII. El usuario introduce su sueldo mensual, el programa calcula el sueldo anual, determina si está exento del impuesto o cuánto debe pagar de ISR y muestra el descuento correspondiente en pantalla.

## Funcionalidades

- Lee el sueldo mensual del empleado.
- Calcula el sueldo anual.
- Determina el ISR según la escala de la DGII 2026.
- Muestra el monto del ISR anual.
- Calcula el descuento mensual del ISR.
- Si el empleado está exento, muestra RD$0.00 o N/A.
